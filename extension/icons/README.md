# ContentCraft AI Extension Icons

This directory should contain the following icon files:

- `icon16.png` - 16x16 icon for browser toolbar
- `icon48.png` - 48x48 icon for extension management page
- `icon128.png` - 128x128 icon for Chrome Web Store

## Icon Design Guidelines

The icons should follow the ContentCraft AI brand:
- Primary color: #8b5cf6 (violet-500)
- Secondary color: #6366f1 (indigo-500)
- Style: Modern, minimalist, AI-themed

## Recommended Icon

A layered/stacked content icon representing AI-generated content:
- Three overlapping document/layer shapes
- Gradient from violet to indigo
- Clean, modern design

You can generate icons using:
1. Figma or Sketch
2. Online icon generators
3. AI image generation tools

Place the generated icons in this directory to complete the extension.
